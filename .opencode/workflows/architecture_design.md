# Workflow: Architecture Design

Executed by the **Architect** agent.

## Steps
1. **Analyze Requirements**: Read the PRD generated by the Product Manager.
2. **Sequential Thinking**: Use the sequential thinking MCP to map out the data flow and component boundaries.
3. **Tech Stack Evaluation**: Compare frontend, backend, database, and deployment options. Score them on: DX, Performance, Community Support, and Licensing.
4. **Interface Design**: Define the core API contracts or CLI arguments.
5. **Documentation**: Write the `Architecture Review` and generate `ADR`s for major decisions (e.g., framework choice, database choice).
